# Simple Editor - かんたん画像編集

シンプルで直感的な画像編集・スタイル変換ツール

## 機能

### 🎨 スタイル変換モード
実写画像を様々なスタイルに変換

- アニメ風
- 水彩画風
- 油絵風
- 漫画風
- 3Dレンダリング風
- ジブリ風
- ピクサー風
- など

### ✏️ 部分編集モード
画像の一部だけを編集

1. ブラシで編集したい箇所を塗る
2. 変更内容を文章で指示
3. 自動で編集完了！

## 使い方

1. WebUIを起動
2. 「かんたん編集」タブを開く
3. モードを選択（スタイル変換 or 部分編集）
4. 画像をアップロード
5. 設定して実行！

## インストール

既に `/extensions/simple-editor/` にインストールされています。
WebUIを再起動すると自動的に読み込まれます。

## カスタマイズ

### スタイルを追加したい場合

`scripts/unified_editor.py` の `STYLES` 辞書に追加：

```python
STYLES = {
    # ... 既存のスタイル ...
    "あなたのスタイル": {
        "prompt": "your style description",
        "negative": "what to avoid"
    },
}
```

### デフォルト設定を変更したい場合

- `steps=20` → ステップ数変更（品質 vs 速度）
- `cfg_scale=7.0` → CFGスケール変更
- `sampler_name="Euler a"` → サンプラー変更

## トラブルシューティング

**Q: タブが表示されない**
A: WebUIを再起動してください

**Q: エラーが出る**
A: モデルが正しく読み込まれているか確認してください

**Q: 変換結果が思った通りにならない**
A: 「強さ」のスライダーを調整してみてください
